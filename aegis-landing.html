Real-World Use

Example Scenarios

AEGIS provides immediate, actionable guidance in the situations that matter most.

🩸 Emergency Medical
User I broke my leg and there is no hospital nearby.
AEGIS 1. Do not move the limb. Immobilise immediately using two straight rigid objects (sticks, boards) as splints. 2. Pad the splint with cloth to prevent pressure sores. 3. Tie firmly above and below the break — not over the fracture site. 4. Check circulation: press the nail until it turns white, release — colour should return in 2 seconds. 5. Elevate and monitor for shock.
☢ Nuclear / Radiation
User Radiation has been detected nearby. What do I do?
AEGIS Get inside immediately. A concrete or brick building provides 10× protection over being outside. Go to the centre of the building, away from windows, on a middle floor. Seal gaps under doors with wet towels. Do not leave for 12–24 hours minimum — radiation falls 90% in the first 7 hours. Remove and bag outer clothing if you were outside.
⚡ Infrastructure Failure
User No electricity. How do I generate power?
AEGIS Immediate: A bicycle generator can produce 50–200W. Connect the rear wheel to an automotive alternator via a belt drive. Short-term: A waterwheel on a stream with 1m head can produce 100–500W continuously — more reliable than wind. Step-by-step construction available in Engineering module. Focus on 12V DC to charge batteries first.
🌱 Food Security
User We have no food and need to grow something fast.
AEGIS Fastest crops from seed: Radishes (25 days), spinach (30 days), lettuce (35 days), beans (50 days). Immediate action: Plant in any container with drainage. Use the worm test — dig 30cm cube, count worms. 10+ means plant now. Fewer than 5: add compost and wait 2 weeks before planting staples.
🔥 Core Survival
User I'm lost in the wilderness. How do I find north?
AEGIS Shadow-tip method: Push a stick into flat ground. Mark shadow tip with stone. Wait 15 minutes. Mark the new tip. The line between stones runs east–west — first stone is west. At night: find the Big Dipper, draw a line through its two outer stars and extend 5× that distance — that is Polaris. Face it to face north.
🏛 Community
User We have a group of survivors and need to organise.
AEGIS First hour: Assign a leader — not the strongest, the person who decides and takes responsibility. First day: Establish resource distribution rules in public before supplies run short. Minimum roles: water, food, security (day/night rotation), medical, shelter. Critical: Quarantine all new arrivals 14 days regardless of apparent health.

Modular Architecture

AEGIS is designed for community contribution. Every knowledge domain is a self-contained module. Adding new knowledge requires no understanding of the AI engine — just write the content.

1
Fork the repository
Clone AEGIS to your GitHub account. No special tools required — a browser and a text editor are enough to contribute knowledge articles.
git clone https://github.com/Aditya-Shaswat/AEGIS
2
Add or edit a knowledge article
Articles are plain Markdown files inside the relevant domain folder. Follow the template: title, summary, step-by-step content, warnings, and sources.
modules/medicine/your-article.md
3
Run the build tool
The build script bundles all modules into a single self-contained index.html file ready for deployment. One command, no dependencies.
python3 tools/build.py
4
Open a Pull Request
Submit your contribution for review. All additions are reviewed for accuracy against primary sources before merging. Every article that ships may save a life.
gh pr create --title "Add [domain]: [article name]"
